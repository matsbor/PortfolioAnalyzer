## TROUBLESHOOTING

### Error: "institutional_enhancements.py not found"
```bash
# Check files in directory
cd ~/PortfolioAnalyzer
ls -la institutional*.py

# If missing, download from Claude
# Make sure you're saving to the RIGHT directory
```

### Error: "No module named 'institutional_enhancements'"
```bash
# Make sure files are in SAME directory as alpha_miner_enhanced.py
cd ~/PortfolioAnalyzer
ls -la

# Both should be in same folder
```

### Integration script doesn't run
```bash
# Try with python3 explicitly
python3 integrate_institutional_v2.py

# Or check Python version
python3 --version
# Should be 3.8 or higher
```

### App starts but looks the same
```bash
# You might be running the OLD file
# Make sure you're running:
streamlit run alpha_miner_institutional_v2.py

# NOT:
streamlit run alpha_miner_enhanced.py
```
